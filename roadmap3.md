# TLA+ Nassi-Shneiderman Roadmap: Phase 3

This document outlines the next level of visual fidelity, semantic analysis, and tool integration for the TLA+ parser visualization project.

## 💎 Visual Excellence & Interactivity
- [ ] **Interactive Diagrams**: Implement Collapse/Expand functionality for `LET-IN`, `CASE`, and `PROOF` blocks to manage complexity in large modules.
- [ ] **Source-to-SVG Linkage**: Clickable blocks that highlight the corresponding line in the structural/source view.
- [ ] **Adaptive Layout**: Replace fixed 800px width with automatic width calculation based on KaTeX formula metrics.
- [ ] **Theme System**: Add "Modern Light", "Nord", and "High Contrast" themes to the Tokyo Night default.
- [ ] **High-Res Export**: CLI option to export diagrams to PNG/PDF via headless browser (Playwright) for inclusion in formal reports.

## 🧠 Semantic & Logic Deconstruction
- [ ] **`EXCEPT` Structural Blocks**: Visualize `[f EXCEPT ![x] = y]` as an explicit update block rather than a flat expression.
- [ ] **Identifier Classification**: Distinct styling/coloring for different symbol kinds in the diagram:
    - `STATE_VAR`: Highlighted as the core of action formulas.
    - `CONSTANT`: Subtler, italicized.
    - `BOUND_VAR`: Visually linked to their quantifier (`\A`, `\E`).
    - `LOCAL_DEF`: Distinct markers for `LET` definitions.
- [ ] **Operator Inlining**: Tooltip preview of operator diagrams when hovering over their calls.
- [ ] **Destructuring Visualization**: Better rendering for tuple/record patterns in binders (`\A <<x, y>> \in S : ...`).

## 🛠️ Advanced Structural Analysis
- [ ] **Call Graph Generation**: Export DOT/Graphviz files showing the dependency graph between operators and modules.
- [ ] **Module `INSTANCE` Container**: Visualizing imported operators as distinct, dashed-border scopes.
- [ ] **Proof Level Indentation**: Improved visual nesting for deep hierarchies (level `<3>` and below).
- [ ] **Temporal Logic Icons**: Dedicated semantic icons for `[]`, `<>`, `~>`, `ENABLED`, `UNCHANGED`.

## 🚀 Integration & Ecosystem
- [ ] **Live Preview Mode**: A `--watch` flag for the CLI to automatically refresh the browser on `.tla` changes.
- [ ] **VS Code Extension**: Embed NSD diagrams directly in the editor sidebar or as a custom editor for `.tla` files.
- [ ] **Documentation Gallery**: A curated gallery showing how standard TLA+ idioms lookup in Nassi-Shneiderman style.
